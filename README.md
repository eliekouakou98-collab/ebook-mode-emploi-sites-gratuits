## ✅ Comment utiliser ce projet

### Étape 1 : Lisez le contenu
- Allez dans `01_CONTENU/contenu_complet_structure.md`
- Lisez tout d'abord, sans rien faire
- Familiarisez-vous avec la structure

### Étape 2 : Capturez les sites
- Suivez les guides dans `02_CAPTURES_SITES/`
- Un guide = un site à capturer
- Sauvegardez vos captures dans `04_IMAGES/`

### Étape 3 : Préparez votre design
- Lisez `03_DESIGN_CANVA/palette_couleurs_complete.md`
- Lisez `03_DESIGN_CANVA/guide_canva_complet.md`
- Commencez à créer sur Canva avec ces spécifications

### Étape 4 : Vérifiez votre travail
- Utilisez `01_CONTENU/checklist_verification.md`
- Avant d'exporter, passez cette checklist

### Étape 5 : Exportez & Publiez
- Exportez depuis Canva en PDF
- Publiez sur Gumroad
- Lancez votre lien !
